# Pulse1x

A lightweight, modern, free utility for monitoring, optimizing, and maintaining your Windows PC.

This repository hosts **compiled releases only** — the source code is closed and not published here.

## Download

See the [Releases](../../releases) page for the latest version of `Pulse1x.App.exe`.

## What's inside the app

- 📊 **Dashboard** — real-time CPU/GPU usage and temperature, RAM, disk, and network throughput
- 🚀 **Optimization** — reversible system tweaks, with a full change history
- ❤️ **Health** — component diagnostics, disk SMART status, and safe cleanup tools
- 🌐 **Latency** — speed/ping testing plus network optimization
- 🧰 **Utility** — Post-Format Center: one-click install of essential apps and drivers via winget
- ☕ A "Buy me a coffee" tab, if the app saved you some time
